# Лабораторная работа №6  
Объектно-ориентированное программирование

## Описание  
Реализована модель оптических дисков с использованием наследования.  
Базовый класс — OpticalDisc, наследники — CD и DVD.  
Каждый класс имеет уникальные поля и методы обработки данных.

---

## Задания варианта  
- Метод 1: вычислить, сколько байт проходит под лазером за **20 секунд**  
  (все дорожки считать равными длине окружности диска).  
- Метод 2: рассчитать **среднее время одного оборота** диска.  
  В DVD учитывается количество слоёв.

Иерархия:
- **CD**
- **DVD**

Реализована перегрузка оператора `+` — сложение возможно только между объектами одного класса.
