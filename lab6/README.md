# Лабораторная работа №6  
**Объектно-ориентированное программирование. Наследование классов (вариант 5)**

## Описание
Реализованы классы для моделирования лодок двух типов:
- весельная лодка,
- моторная лодка.

Используется наследование, переопределение методов и перегрузка оператора `+`.

---

## Классы

### 1. `Boat`
Базовый класс. Свойства:
- название,
- максимальная скорость,
- вместимость.

Методы:
- `info()` — вывод информации,
- `__add__` — комбинирование двух лодок (сумма скоростей и вместимости).

### 2. `PaddleBoat` (наследник Boat)
Дополнительные поля:
- количество вёсел.

Переопределён метод `info()`.

### 3. `MotorBoat` (наследник Boat)
Дополнительные поля:
- мощность двигателя.

Переопределён метод `info()`.

---

## Перегрузка оператора `+`
Операция `boat1 + boat2` создаёт объект `CombinedBoat`  
с суммарной скоростью и вместимостью.
