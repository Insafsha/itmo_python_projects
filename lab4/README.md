# Лабораторная работа №4  
**Задача о рюкзаке – инвентарь Тома (вариант 5)**

##  Условие  
Том должен собрать инвентарь для марш-броска к радиовышке.  
Размер рюкзака: **2×4 клетки (8 ячеек)**  
Стартовые очки: **20**  
Болезнь: **астма → ингалятор обязателен**  

Каждый предмет:  
- занимает определённое число ячеек  
- даёт (или отнимает) очки выживания  

Необходимо:  
- подобрать набор предметов  
- уложиться в объём  
- включить обязательный предмет  
- чтобы итоговые очки > 0  
- вывести рюкзак в виде сетки
