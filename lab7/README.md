# Лабораторная работа №7  
Работа с API 

## Описание  
В работе реализовано три независимых задания, каждое использует внешний API:

---

## 1. Погода (OpenWeatherMap)
Получает информацию о городе:

- описание погоды  
- температуру  
- ощущаемую температуру  
- влажность  
- давление  

Данные приходят в формате JSON и форматируются в человекочитаемый вид.

---

## 2. Dictionary API (dictionaryapi.com)
Программа получает информацию о слове:

- часть речи  
- произношение  
- определения (shortdef)  
- примеры использования, если доступны  

Все данные извлекаются из вложенных структур JSON.

---

## 3. Дополнительное задание — Tkinter + RandomFox API
Графическое приложение, которое:

- загружает случайную картинку лисы через API  
- отображает её в окне  
- по нажатию кнопки загружает новую картинку  

Используются библиотеки `tkinter`, `PIL`, `requests`.
