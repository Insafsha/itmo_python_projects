# Лабораторная работа №3  
Генератор ключей на Tkinter (Вариант 5)

##  Описание  
Приложение создаёт keygen для ПО.  
Пользователь вводит первый блок ключа (5 символов A–Z, 0–9), дальше генерируются ещё 2 блока:

- **2 блок** — сдвиг каждого символа на **3 позиции вправо**
- **3 блок** — сдвиг каждого символа на **5 позиций влево**

Финальный формат:

```
XXXXX-XXXXX-XXXXX
```
