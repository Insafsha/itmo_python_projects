# Лабораторная работа №9

## Создание веб-приложения на Flask с базой данных

### Описание

Приложение представляет собой простую систему отзывов.  
Пользователь может:

- оставить текстовый отзыв
- выбрать оценку от 1 до 5
- увидеть все ранее оставленные отзывы
- (допзадание) очистить ленту сообщений одной кнопкой

Для хранения данных используется база SQLite, работающая через Flask-SQLAlchemy.  
Интерфейс оформлен с помощью HTML + CSS: добавлен фон, шапка страницы, стиль карточек отзывов и окрашенные звёзды.

### Основная функциональность

- Приём POST-запроса с формы
- Создание записи `Review(text, rate)` в базе данных
- Автоматическое создание таблиц при первом запуске
- Вывод всех отзывов на страницу
- Отображение рейтинга в виде звёзд

### Используемые технологии

- Flask
- Flask-SQLAlchemy
- HTML / CSS (собственные шаблоны)
- SQLite (автоматически создаётся файл `reviews.db`)

### Дополнительное задание

Реализована кнопка **Clear**, которая удаляет все отзывы из базы и обновляет страницу.
