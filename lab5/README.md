# Лабораторная работа №5  
Работа с регулярными выражениями (Regex)

В данной лабораторной работе решаются три основных задания с использованием регулярных выражений, а также дополнительная задача по поиску скрытых данных.

---

##  Задание 1  
Открыть файл **task1-en.txt** (или task1-ru.txt) и выполнить поиск согласно варианту.

**Вариант 5:**  
- найти все числа (целые и дробные);  
- найти все слова длиной **6** и **8** букв.

Используемые регулярные выражения:  
- числа: `\b(0|[1-9]\d*)(?:\.\d+)?\b`  
- слова 6 или 8 букв: `\b\w{6}\b|\b\w{8}\b`

---

##  Задание 2  
Открыть файл **task2.html** и выполнить поиск согласно варианту.

**Вариант 5:**  
- извлечь строки, находящиеся в атрибуте **content=""** любого тега.  

Регулярное выражение:  
- `content="([^"]+)"`

---

##  Задание 3  
Открыть файл **task3.txt**, содержащий перемешанные данные пользователей:  
ID, фамилия, email, дата регистрации, сайт — в случайном порядке.

Необходимо:  
1. извлечь данные каждого типа регулярными выражениями;  
2. восстановить таблицу в нормальной форме;  
3. сохранить результат в файл **task3_result.csv**.

Используемые выражения:  
- Сайт: `https?://[a-zA-Z0-9.-]+/`  
- Дата: `\d{4}-\d{2}-\d{2}`  
- Фамилия: `[A-Z][a-z]+(?!\d\d@|@)`  
- Email: `[a-z][a-z0-9-]*@[a-z0-9-]+\.[a-z]{2,}`  

---

##  Дополнительное задание  
В файле **task_add.txt** спрятаны:  
- 5 дат в разных форматах,  
- 5 email-адресов,  
- 5 адресов сайтов.  

Каждый элемент скрыт в “белом шуме” и предваряется пробелом.  
Необходимо найти все 15 элементов с помощью регулярных выражений.
