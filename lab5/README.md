# Лабораторная работа №5  
**Работа с регулярными выражениями **

## Описание  
В работе выполняются три задания по извлечению данных из текстов и HTML с помощью регулярных выражений, а также дополнительная задача.  
Каждое задание реализовано отдельным модулем, запуск объединён в `main.py`.

## Задание 1  
Обработка текстового файла.

**Вариант 5:**  
- найти все числа (целые и дробные);  
- найти все слова длиной **6** и **8** букв.

**Используемые регулярные выражения:**  


\b\d+(?:.\d+)?\b
\b[A-Za-zА-Яа-я]{6}\b
\b[A-Za-zА-Яа-я]{8}\b


Результат → `results/task1_result.txt`

---

## Задание 2  
Извлечение строк из атрибутов `content="..."` в HTML-файле.

**Регулярное выражение:**  


content="([^"]+)"


Результат → `results/task2_result.txt`

---

## Задание 3  
Нормализация сырых данных: выделить и собрать таблицу из  
**ID, Surname, Date, Email, Site**.

**Регулярные выражения:**  


Дата: \d{4}-\d{2}-\d{2}
Email: [a-z0-9._-]+@[a-z0-9.-]+.[a-z]{2,}
Сайт: https?://[\w.-]+(?:/[\w./-]*)?
Фамилия: \b[A-Z][a-z]+\b


Результат → `results/task3_result.csv`

---

## Дополнительное задание  
Поиск данных, идущих после пробела:

- 5 дат  
- 5 email  
- 5 сайтов  

**Регулярные выражения:**  


Дата: (?<= )\d{4}[-/.]\d{2}[-/.]\d{2}
Email: (?<= )[a-z0-9._-]+@[a-z0-9.-]+.[a-z]{2,}
Сайт: (?<= )https?://[\w.-]+(?:/[\w./-]*)?


Результат → `results/task_add_result.txt`
