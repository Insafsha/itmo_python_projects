# Лабораторная работа №8  
Работа с OpenCV (Вариант 5)

## 1. Преобразование изображения  
По условию варианта требуется добавить шум к исходному изображению.

Выполнено:  
- изображение варианта №5 загружается с диска;  
- применяется гауссов шум (σ = 0.1);  
- результат отображается на экране;  
- создаётся файл `variant-5-noisy.jpg`.

Использованы: OpenCV, NumPy.

---

## 2. Отслеживание метки на видео  
Используется камера и цветовая сегментация.

Реализовано:  
- захват кадра с веб-камеры;  
- преобразование в HSV;  
- выделение синей области;  
- поиск контура метки;  
- нахождение центра (cx, cy);  
- отрисовка центра и обводки.

Кадр обновляется в реальном времени.

---

## 3. Модификация программы (вариант №5)  
По условию необходимо изменить цвет обводки метки:

- если центр метки попадает в **левый верхний угол** (50×50) → рамка синяя;  
- если центр в **правом нижнем углу** → рамка красная;  
- в остальных случаях рамка зелёная.

Цвет обводки меняется динамически при движении метки.

---

## Дополнительное задание  
Наложить изображение `fly64.png` поверх кадра так, чтобы  
центр мухи совпадал с центром метки.

Выполнено:  
- PNG загружается один раз при старте программы;  
- вычисляется позиция размещения;  
- выполняется альфа-смешивание;  
- муха «привязана» к движению метки в реальном времени.

---

Работа охватывает базовую обработку изображений, фильтрацию, цветовые пространства, поиск объектов и наложение графики в видеопотоке.
