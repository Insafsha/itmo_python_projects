# Лабораторная работа №1  
**Визуализация в консоли и анализ числовой последовательности**

##  Описание  
В этой работе реализованы несколько самостоятельных мини-задач, направленных на закрепление базовых конструкций Python:

- вывод цветных паттернов с помощью escape-последовательностей;  
- текстовая визуализация графика функции;  
- построение симметричной фигуры при помощи циклов;  
- чтение данных из файла и вычисление статистических показателей;  
- построение простой вертикальной диаграммы в консоли.

Каждая задача вынесена в отдельный модуль для удобства чтения и повторного запуска.

---

##  Структура

```
lab1/
├── flag_lithuania.py        # вывод флага Литвы (ANSI escape codes)
├── abs_graph.py             # график функции y = |x| (текстовая визуализация)
├── pattern_figure.py        # построение симметричной фигуры
├── sequence_analysis.py     # анализ чисел из файла и вывод диаграммы
├── sequence.txt             # входные данные (250 чисел)
└── README.md
```

---

##  Задачи

### **1. Флаг Литвы**
Генерация цветных полос с использованием ANSI-кодов. Реализован отдельный рендер для каждой полосы.

---

### **2. График функции `y = |x|`**
Текстовая визуализация графика в первой четверти.  
Используется фиксированный шаг и псевдо-пиксели на основе пробелов и цветных блоков.

---

### **3. Симметричная фигура**
Построение геометрической фигуры с использованием вложенных циклов и форматирования отступов.

---

### **4. Анализ числовой последовательности**
Файл `sequence.txt` содержит 250 значений.  
Сценарий выполняет:

- загрузку данных;  
- преобразование значений в абсолютные;  
- разделение по чётным/нечётным индексам;  
- вычисление среднего значения в каждой группе;  
- построение вертикальной блок-диаграммы из белых прямоугольников;  
- сравнение усреднённых значений групп.

---

##  Запуск

Запуск каждого модуля осуществляется отдельно:

```bash
python flag_lithuania.py
python abs_graph.py
python pattern_figure.py
python sequence_analysis.py
```
