Лабораторная работа №1

Визуализация в консоли и анализ числовой последовательности

Описание

В этой лабораторной работе реализовано несколько мини-задач, направленных на практику базовых конструкций Python:

использование escape-последовательностей для вывода цветных паттернов в консоль;

рисование графика функции при помощи текстовой визуализации;

вывод симметричной фигуры через циклы и форматирование строки;

чтение и анализ числовой последовательности из файла и построение простой процентной диаграммы.
