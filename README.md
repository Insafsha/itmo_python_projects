# Учебные мини-проекты и практические работы по Python

Репозиторий содержит 10 полноценных мини-проектов, отражающих навыки работы с алгоритмами, файлами, сетью, API, визуализацией, обработкой данных, компьютерным зрением, веб-разработкой и голосовыми интерфейсами.

Каждая лабораторная — самостоятельный проект со своей архитектурой, документацией и результатом.

##  Используемые технологии

- **Python 3**
- **Регулярные выражения (re)**
- **Tkinter**
- **OpenCV (cv2)**
- **Flask + SQLAlchemy**
- **Работа с файлами, CSV, XML**
- **API:** OpenWeather, DictionaryAPI, RandomUser, Rick&Morty, NASA, RandomFox, VOSK
- **Модели распознавания речи VOSK**
- **Pillow (PIL)**
- **Глубокая работа с изображениями**
- **Сетевые запросы: requests**
- **SQL, SQLite**
- **pyttsx3 (TTS)**
- **PyAudio (audio stream)**
