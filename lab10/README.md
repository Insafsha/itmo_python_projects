# Лабораторная работа №10
## Голосовой ассистент на основе VOSK

### Описание

Создан голосовой ассистент, работающий на английском языке и использующий локальное распознавание речи через VOSK.  
Ассистент принимает команды голосом, обрабатывает слово через публичный словарь API, отвечает синтезом речи и может сохранять данные локально.

### Основные компоненты:

- **Vosk + PyAudio** — распознавание речи с микрофона
- **pyttsx3** — озвучивание ответов
- **Dictionary API** — получение информации о словах
- **webbrowser** — открытие ссылки с определением
- **локальное сохранение** — запись результата в файл

### Возможности ассистента

| Команда | Действие |
|---------|----------|
|  `find <word>` | Загружает данные о слове с dictionaryapi.dev |
|  `meaning` | Произносит определение слова |
|  `example` | Озвучивает пример использования |
|  `link` | Открывает словарную страницу в браузере |
|  `save` | Сохраняет слово и его определение в файл `dictionary_saved.txt` |
|  `exit` / `stop` | Завершает работу |

### Архитектура

Ассистент состоит из трёх классов:

- **Speech** — синтез речи
- **Recognizer** — потоковое распознавание голоса
- **DictionaryAssistant** — логика команд и взаимодействие с API

Для работы Vosk используется локальная модель `model_en`.

### Особенности

- работает без интернета для распознавания голоса (модель локальная)
- понимает фразы в свободной форме: “find apple”
- корректно обрабатывает ошибки API
- выводит текст в консоль и озвучивает голосом
- структура проекта соответствует требованиям ассистента реального времени
